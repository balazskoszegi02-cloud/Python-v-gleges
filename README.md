Eredetileg grafikus menüt terveztem Tkinter modullal, de inkább parancssorosra váltottam.

# Főbb funkciók
Új edzés rögzítése: gyakorlat neve, ismétlésszám, súly. utóbbi csak szám lehet - különben hiba  
Napló megtekintése: A text fileba rögzített adatokat beolvassa és kiírja a program  
Az edzéseket a program text fájlba menti, dátummal.  

# fájlok
